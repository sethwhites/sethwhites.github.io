# Server Project
I want to create a server that is able to host a variety of different apps that I can access remotely.

## Apps:
  like google photos
  Jellyfin
  others I can't think of off the top of my head
  thing to support vpn
  reverse proxy

## Build:
### Parts:
  - HPZ440 Workstation
  - 4x 26TB Seagate HDD (Refurbished)
  - 6x 8GB ECC RAM
  - Upgrade CPU to E5-2690 v4 2.60GHz
  - [2 x 5.25" to 3 x 3.5" Adapter](https://www.amazon.com/dp/B07YQKHL8N?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
  - [SSD for OS](https://www.amazon.com/dp/B08B3JBDJ6?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
  - NVIDIA Quadro GPU

### Installation:
- ubdate HPZ440 BIOS
- get truenas [disc image](https://www.truenas.com/download/)
- write disc image to USB using [RUFUS](https://rufus.ie/en/)
- boot off of USB
- install truenas
- configure and start installing apps
